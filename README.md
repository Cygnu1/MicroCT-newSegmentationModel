# microct-newSegmentationModel
Implementation of the new method for rock micro-CT segmentation only (preprocessing + Segmentation + porosity)

---

## Overview

This project presents a computational pipeline for image processing and analysis using
state-of-the-art techniques in computer vision and machine learning.

This project is a new segmentation model for the segmentation of porous in microCT. Specialist done new ground-truth, a better one, so a new method is required

---

## Repository Structure
There's a script for each technique and model. (CLAHE, HM, TRAIN, EVAL, POROSITY and ERROR)

---

## Dataset
The datasets used in this work are not publicly available due to size and licensing
constraints. Instructions for reproducing the experiments are provided.

The requirements for the virtual env is available.
